# 🚀 VoltAI Chat App

A full-stack AI-powered chat application built using **.NET Core**, **Angular**, and **Groq API**.
This app allows users to interact with an AI assistant in real-time through a clean and responsive UI.

---

## 🧠 Features

* 💬 Real-time AI chat interaction
* ⚡ Fast responses using Groq API
* 🧩 Full-stack architecture (Angular + .NET Core)
* 📡 REST API integration
* 🎨 Clean and responsive UI
* 🔒 Secure API handling (no keys exposed)

---

## 🛠️ Tech Stack

### 🔹 Frontend

* Angular
* TypeScript
* HTML / CSS

### 🔹 Backend

* .NET Core Web API
* C#
* Entity Framework Core

### 🔹 AI Integration

* Groq API (LLaMA models)

## ⚙️ Setup Instructions

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

---

### 2️⃣ Backend Setup (.NET)

```
cd backend
dotnet restore
dotnet run
```

---

### 3️⃣ Frontend Setup (Angular)

```
cd frontend
npm install
ng serve
```

---

## 🔐 Environment Variables

Create a local configuration for your API key:

```
GROQ_API_KEY=your_api_key_here
```

⚠️ Never expose your API key in public repositories.

---

## 🌐 API Flow

```
User → Angular UI → .NET API → Groq API → Response → UI
```
